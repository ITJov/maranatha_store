🛍️ Maranatha Store – Web E-Commerce Application

Maranatha Store adalah aplikasi e-commerce berbasis web yang dikembangkan sebagai project akademik untuk mensimulasikan sistem penjualan online dengan dua role utama: Admin dan Customer. Aplikasi ini mencakup manajemen produk, transaksi, pembayaran, dan pengelolaan user dalam satu sistem terintegrasi.

🚀 Tech Stack

- Laravel
- PHP
- MySQL
- Bootstrap / CSS
- MVC Architecture

👥 User Roles
🛠️ Admin Page

Admin memiliki akses untuk:
- CRUD Produk
- CRUD Kategori
- Manajemen User & Role
- Melihat & Mengelola Order
- Mengelola Invoice & Payment
- Monitoring transaksi

🛒 Customer Page

Customer dapat:
- Registrasi & Login
- Melihat daftar produk
- Menambahkan produk ke cart
- Checkout & melakukan pembayaran
- Melihat riwayat transaksi

🧩 Fitur Utama

- 🔐 Authentication & Role-based Access
- 🛒 Shopping Cart System
- 💳 Checkout & Invoice
- 📦 Manajemen Produk & Kategori
- 🧾 Sistem Order & Payment
- 🛡️ Validasi input & proteksi duplicate entry
